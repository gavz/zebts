# zebts
Ziggy's Embedded BTS
***

An armhf image featuring:

* Asterisk
* OpenBTS
* Yate
* YateBTS
* Osmo-TRX
* Smqueue
* Sipauthserve
* AirSpy

for HackRF, BladeRF, or USRP.

Also included are the patches & scripts used to build the debian packages and create the armhf image.
